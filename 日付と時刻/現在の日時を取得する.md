現在の日時を取得する
====================

現在の日時を取得するコードは、つぎのようになります。

```haskell
--- time
import Data.Time (getCurrentTime)

main :: IO ()
main = do
	ct <- getCurrentTime
	print ct
```

ここで使われている関数getCurrentTimeは、つぎのような型を持ちます。

```haskell
getCurrentTime :: IO UTCTime
```

現在の時刻をUTCで取得します。
型UTCTimeは、つぎのようなフィールドを持ちます。

```haskell
data UTCTime = UTCTime {
	utctDay :: Day,
	utctDayTime :: DiffTime
	}
```

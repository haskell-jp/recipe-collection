最大公約数と最小公倍数を求める
==============================

最大公約数
----------

最大公約数を求めるには関数gcdを使います。

```haskell
> gcd 18 42
6
```

関数gcdの型は、つぎのようになります。

```haskell
gcd :: Integral a => a -> a -> a
```

最小公倍数
----------

最小公倍数を求めるには関数lcmを使います。

```haskell
> lcm 18 42
126
```

関数lcmの型は、つぎのようになります。

```haskell
lcm :: Integral a => a -> a -> a
```

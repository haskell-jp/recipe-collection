対数計算を行う
==============

自然対数
--------

自然対数を求めるには関数logを使います。

```haskell
> log 10
2.302585092994046
```

関数logの型は、つぎのようになります。

```haskell
log :: Floating a => a -> a
```

底を指定
--------

底を指定して、対数を求めるには関数logBaseを使います。

```haskell
> logBase 10 10000
4.0
> logBase 2 128
7.0
```

関数logBaseの型は、つぎのようになります。

```haskell
logBase :: Floating => a -> a -> a
```

ネイピア数のべき乗
------------------

ネイピア数のべき乗を求めるには関数expを使います。

```haskell
> exp 1
2.718281828459045
> exp 3
20.085536923187668
```

関数expの型は、つぎのようになります。

```haskell
exp :: Floating a => a -> a
```

べき乗を求める
==============

整数乗
------

べき乗を求めるには演算子(^)を使います。

```haskell
> 3 ^ 8
6561
```

3を8乗しています。
演算子(^)の型は、つぎのようになります。

```haskell
(^) :: (Num a, Integral b) => a -> b -> a
```

実数乗
------

実数による、べき乗を求めるには演算子(\*\*)を使います。

```haskell
> 3 ** 8
6561.0
> 2 ** (1 / 2)
1.4142135623730951
```

演算子(\*\*)の型は、つぎのようになります。

```haskell
(**) :: Floating a => a -> a -> a
```
